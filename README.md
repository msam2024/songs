# وب‌اپ اشعار فارسی (نمونه)

این یک وب‌اپ ایستا (HTML/CSS/JS) برای نمایش خوانندگان و اشعارشان به زبان فارسی است. ویژگی‌ها:
- رابط کاملاً فارسی و راست‌چین (RTL).
- فهرست خوانندگان با زیربخشِ فهرست آهنگ‌ها و نمایش کامل متن شعر.
- جستجوی نام خواننده و عنوان آهنگ.
- فرم افزودن شعر جدید.
- کیبورد فارسی مجازی برای تایپ مستقیم حروف فارسی.
- ذخیره‌سازی در localStorage مرورگر (بدون نیاز به سرور).

اجرای محلی:
1. فایل‌ها را در یک پوشه قرار دهید.
2. فایل `index.html` را در مرورگر باز کنید.

اگر می‌خواهید این پروژه را در گیت‌هاب قرار دهم، نام مخزن را به صورت `owner/repo` ارسال کنید تا من فایل‌ها را به آن مخزن پوش کنم. همچنین می‌توانم شاخهٔ جدید بسازم یا در ریشهٔ مخزن فعلی قرار دهم — لطفاً مشخص کنید.
